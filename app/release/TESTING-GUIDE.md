## 🚀 安装方法

### 方法 1: ADB 安装（推荐）
```bash
# 连接 Android 设备
adb devices

# 卸载旧版本（如果有）
adb uninstall com.sat20.wallet

# 安装新版本
adb install /Users/icehugh/workspace/jieziyuan/client/sat20wallet/app/release/SAT20-Wallet-v0.1.12-release-final.apk

# 验证安装
adb shell pm list packages | grep sat20
```

### 方法 2: 直接安装
1. 将 APK 发送到手机
2. 在文件管理器中点击 APK
3. 允许"未知来源"安装
4. 完成安装

### 方法 3: AirDrop/云盘
1. 通过 AirDrop 发送到 iPhone（需要用 Android 设备中转）
2. 或上传到 Google Drive/Dropbox
3. 在 Android 设备下载并安装
